# Yanhekt Downloader v0.0.3

安装体验与下载完成收尾修复版。

## 下载

- `YanhektDownloader_Setup_v0.0.3.exe`

## 修复

- 修复安装器勾选“创建桌面快捷方式”后，部分 Windows 机器上桌面没有出现快捷方式的问题。
- 快捷方式创建不再依赖 PowerShell，避免 PowerShell/.NET 环境异常导致静默失败。
- 安装器会读取 Windows 真实桌面位置，兼容 OneDrive 或系统策略重定向后的桌面路径。
- 修复 GUI 下载成功后仍停留在下载窗口的问题；下载完成后会短暂提示并自动退出。
- 下载任务结束时会等待并清理本工具启动的专用 Chrome 进程，减少后台残留。
