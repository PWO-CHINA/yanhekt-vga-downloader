# yanhekt/延河课堂 Downloader v0.0.10

这是第三个下载稳定性加固版，重点减少“自己的电脑能下，同学电脑加载课程成功但下载阶段没进展或失败”的情况。

## 下载

- `YanhektDownloader_Setup_v0.0.10.exe`

## 改进

- ffmpeg 下载时会使用本工具专用 Chrome/Edge 登录会话里的真实浏览器 `User-Agent`。
- 下载前会临时读取媒体相关域名的 Cookie，并传给 HLS 清单预检与 ffmpeg。Cookie 不写入日志，不保存到配置文件，只在当前下载进程内使用。
- 下载前估算大小后，会在真正启动 ffmpeg 前重新生成一次视频签名地址，降低慢电脑或慢网络下签名过期的概率。
- HLS 预检会同时检查 init map / key 等资源和第一个视频分片，减少 ffmpeg 启动后才暴露问题的情况。
- 源码命令行模式下进度输出更稳定；GUI 模式仍会收到逐行进度。
- 安装器会验证随包携带的 `ffmpeg.exe -version` 能正常运行，提前发现被杀毒软件隔离、损坏或系统策略拦截的问题。
