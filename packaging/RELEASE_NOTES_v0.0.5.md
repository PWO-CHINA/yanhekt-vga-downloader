# yanhekt/延河课堂 Downloader v0.0.5

可靠性改进版，重点减少同学电脑上“安装了但打不开、点了没反应、下载到一半才失败”的情况。

## 下载

- `YanhektDownloader_Setup_v0.0.5.exe`

## 改进

- 默认保存目录改为用户下载文件夹下的 `YanhektDownloader`，避免安装目录不可写导致下载失败。
- 下载前会检查保存目录是否可写，并对过深路径给出明确提示。
- 勾选“下载前估算占用空间”时，会在下载前检查保存目录所在磁盘剩余空间，不足时提前停止。
- 自动压缩过长文件名，降低 Windows 路径长度和 ffmpeg 写入失败的概率。
- 文件名会避开 Windows 保留设备名，例如 `CON.mp4`、`AUX.mp4`。
- 打包版 GUI 如果缺少 `YanhektDownloaderWorker.exe`，会明确提示安装不完整，不再静默退回源码脚本。
- 后台任务失败时，GUI 会弹出更具体的中文提示，覆盖缺浏览器、缺 ffmpeg、需要登录、课程链接不对、保存目录不可用等常见情况。
- GUI 崩溃日志优先写入 `%LOCALAPPDATA%\YanhektDownloader\yanhekt_gui_error.log`，安装到只读目录时也更容易看到错误原因。
- 安装器会检查临时目录和安装目录所在磁盘空间，校验主程序、Worker、ffmpeg、README、LICENSE、VERSION 是否完整。
- 安装后启动失败或桌面快捷方式创建失败时，安装器会明确提示，不再只显示“安装完成”。
- 不再自动扫描主 Chrome / Edge 用户数据目录里的 DevTools 端口；默认只使用本工具专用浏览器配置，或用户显式传入的 `--cdp`。
