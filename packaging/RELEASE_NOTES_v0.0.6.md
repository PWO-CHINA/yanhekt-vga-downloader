# yanhekt/延河课堂 Downloader v0.0.6

回归修复版，重点修复 v0.0.5 在点击“开始下载勾选项”后可能没有日志、没有进度、看起来卡住的问题。

## 下载

- `YanhektDownloader_Setup_v0.0.6.exe`

## 修复

- 修复专用浏览器配置目录里残留旧 `DevToolsActivePort` 时，下载阶段可能先连接失效端口而卡住的问题。
- 复用专用浏览器端口前会先做快速连通性检查；失效时只清理本工具专用目录里的旧端口文件，并重新启动专用浏览器。
- Worker 的关键状态输出改为立即刷新，GUI 能更早看到“正在读取课程清单”“正在准备视频地址”“正在估算占用空间”等状态。
- GUI 在后台任务启动后会进入活动进度状态，避免长步骤期间看起来完全没反应。
- 浏览器启动参数增加对 Chrome / Edge 自动化提示条的抑制，降低第一次登录时看到 DevTools 相关提示造成误解的概率。
