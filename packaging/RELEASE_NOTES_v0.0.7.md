# yanhekt/延河课堂 Downloader v0.0.7

这是一个下载兼容性修复版，重点处理部分同学电脑上课程清单能加载、但下载到第一个 `.ts` 分片时 ffmpeg 报 `Invalid data found when processing input` 的问题。

## 下载

- `YanhektDownloader_Setup_v0.0.7.exe`

## 修复

- 下载前会把远端 HLS 清单临时改写为本地清单，让 ffmpeg 读取带签名参数的完整分片地址，避免某些网络/CDN 节点拒绝未签名分片。
- 调用 ffmpeg 前新增视频清单与第一个分片预检。如果拿到的是登录页、错误页、拦截页或非媒体数据，会给出更明确的中文原因和建议。
- ffmpeg 与 Python 预检使用更一致的媒体请求头，降低同一链接“估算大小成功、实际下载失败”的概率。
- ffmpeg 错误日志里的媒体签名链接会脱敏，方便截图反馈时降低 token/签名泄露风险。
- GUI 增加“正在检查视频分片访问”状态，并对分片访问失败给出更具体的提示。
