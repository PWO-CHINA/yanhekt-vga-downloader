# yanhekt/延河课堂 Downloader v0.0.8

这是一个主动稳定性加固版，目标是在更多同学的 Windows 电脑、Edge/Chrome 环境和校园网络里减少“能加载课程但下载失败”的情况。

## 下载

- `YanhektDownloader_Setup_v0.0.8.exe`

## 稳定性加固

- HLS 清单改写现在会区分视频分片、子清单、key 和 init-map，避免把 `#EXT-X-KEY` 或 `#EXT-X-MAP` 当成第一个视频分片检测。
- 遇到 HLS 主清单时，会优先选择带视频编码或分辨率的视频变体，降低误选音频-only 或不合适变体的概率。
- 下载前估算大小与 ffmpeg 实际下载使用同一套 HLS 变体选择逻辑，进度和磁盘预检更一致。
- ffmpeg 增加网络重连参数和无进展 watchdog；长时间没有进展时会终止并给出明确提示，不再让 GUI 看起来一直卡住。
- 单节课下载失败时，会自动重新签名视频地址并重试一次，覆盖临时网络抖动、短时 CDN 问题或签名过期。
- ffmpeg 成功退出后会再次检查最终 `.mp4` 是否完整，避免空文件或损坏文件被误报为下载完成。
- 如果专用浏览器已经打开但登录过期，程序会主动打开可见登录窗口，而不是直接失败。
- 安装器覆盖安装前会检测旧版程序或 Worker 是否仍在运行，提醒先关闭后再安装。
- GUI 增加浏览器启动失败、下载无进展、mp4 不完整等更具体的中文提示。
