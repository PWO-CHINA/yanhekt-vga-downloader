# yanhekt/延河课堂 Downloader v0.0.9

这是第二个主动加固版，重点放在“出问题时更容易反馈和恢复”，不改变 v0.0.8 的核心下载链路。

## 下载

- `YanhektDownloader_Setup_v0.0.9.exe`

## 改进

- GUI 每次加载课程清单或开始下载都会保存完整任务日志。
- 任务失败弹窗会显示日志文件路径，方便同学反馈完整日志，不再只能截图最后几行。
- 任务日志默认保存在 `%LOCALAPPDATA%\YanhektDownloader\logs`。
- 日志路径处理更稳，即使 GUI 处于异常状态也不会因为缺少日志路径而二次报错。
